# Milestone v1.0: SEO & Content Enhancement

**Status:** SHIPPED 2026-01-27
**Phases:** 1-7
**Total Plans:** 19

## Overview

Transform Maxa's basic test download pages into rich, SEO-optimized study resources. We start with technical SEO foundation (sitemap, robots, metadata), add structured data for rich snippets, integrate interactive normering visualizations, create Swedish strategy content hub, and complete with cross-linking and polish. Each phase builds on the previous to establish Maxa as the authority on Hogskoleprovet-related searches.

## Phases

**Goal**: Search engines can efficiently crawl and index all pages with proper metadata
**Depends on**: Nothing (first phase)
**Plans**: 2 plans

Plans:
- [x] 01-01: Create sitemap.xml and robots.txt

**Key Decisions:**
- metadataBase set to https://maxa.se for URL resolution
- Static JSON for normering data (SSG-compatible, no runtime DB calls)
- Article schema over ScholarlyArticle per Google requirements
- Swedish du-tilltal for authoritative but friendly content
- Progressive enhancement: table-first, chart when JS loads
- Same-season priority in related tests algorithm
- Dark theme OG images matching site branding
